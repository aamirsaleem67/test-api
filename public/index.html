API Service

Welcome to Muhammad Aamir's API service. This is a demonstration of hosting APIs with Cloudflare.

Cloudflare Functions Endpoint

Try the Cloudflare Functions API endpoint: /api/hello

Example response:

{
  "message": "Hello from muhammad-aamir.com API!",
  "timestamp": "2024-05-06T10:00:00.000Z",
  "version": "1.0.0"
}

NestJS API Endpoints

Access your NestJS API through the proxy: /api/nest/[endpoint]

For example: /api/nest/users will proxy to your NestJS server's /users endpoint.

Hybrid Architecture

This site demonstrates a hybrid architecture:

  • Cloudflare Pages & Functions: Hosts static content and simple API endpoints
  • NestJS on Node.js host: Powers your full backend application
  • Cloudflare as proxy: Routes requests between the two systems

To make this work, deploy your NestJS app to a Node.js host and update the NEST_API_URL in functions/_middleware.js.

Built with Cloudflare Pages Functions & NestJS.
